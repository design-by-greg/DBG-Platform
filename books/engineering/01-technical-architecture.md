# 01 Technical Architecture

Status: Draft v1.0

DBG Platform must keep business logic separate from interface logic.

## Layers

Domain Model
Business Rules
Engines
Interfaces
Connectors
Storage

## Interfaces

WordPress is the public interface.
WooCommerce handles checkout.
Base44 handles operations.
GitHub is the documentation source.
Canva is the presentation layer.

## Rule

Technology serves the business domain. The domain must not depend on technology.
