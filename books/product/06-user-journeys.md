# 06 User Journeys

Status: Draft v1.0

## First order

User creates or joins an Organisation.
User creates a Project.
User selects a product or asks for help.
User uploads or selects an Asset.
User validates BAT.
Order enters Production.
Product becomes reorderable.

## Reorder

User opens Product Library.
User selects a validated product.
User chooses quantity or variation.
Order is prepared without restarting configuration.

## Project management

User opens a Project and sees Assets, orders, documents, status and history.
