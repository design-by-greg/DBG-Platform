# 01 Executive Summary

Status: Draft v1.0

DBG Platform is a product built by Design by Greg under the internal project name Project Atlas.

Its goal is to help organisations manage visual communication projects, reusable assets, production flows and reorder history from one platform.

The product is not only an online printing catalogue. It is a project and asset management platform specialised for visual communication.

## Core idea

Organisations do not only buy products. They build a graphic heritage over time.

## Target users

Companies, associations, clubs, public bodies, partners and Design by Greg internal teams.

## MVP modules

Identity Engine
Project Engine
Asset Engine
Commerce Engine
Workflow Engine
Production Engine
