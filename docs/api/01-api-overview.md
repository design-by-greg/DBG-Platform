# 01 API Overview

Status: Draft v1.0

## Purpose

The API exposes DBG Platform business capabilities to interfaces such as WordPress, WooCommerce, Base44 and future mobile or SaaS clients.

## Principles

- Domain-first endpoints
- Organisation scoped data
- Project-first workflows
- Asset-first reuse
- Supplier complexity hidden from public interfaces
- Auditability for important actions

## Main API groups

- Identity API
- Project API
- Asset API
- Commerce API
- Workflow API
- Production API

## Rule

The API must remain stable even if WordPress, Base44 or supplier systems change.
