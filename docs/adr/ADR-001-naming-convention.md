# ADR-001 — Naming Convention

**Status:** Accepted  
**Project:** Project Atlas  
**Product:** DBG Platform  
**Company:** Design by Greg

## Context

Le projet doit distinguer clairement l'entreprise, le produit logiciel et le chantier interne de développement.

## Decision

- **Design by Greg** désigne la société et la marque commerciale.
- **DBG Platform** désigne le produit logiciel.
- **Project Atlas** désigne le projet interne de conception et développement.

## Consequences

Cette convention sera utilisée dans tous les documents, dépôts, maquettes, tickets, modules et communications internes.

## Impacted areas

- Product Blueprint
- GitHub
- Canva
- Documentation
- Architecture
